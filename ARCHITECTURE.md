# Architecture Map — autopus-adk

> Auto-generated by `/auto setup`. Do not edit manually.

## Overview

Autopus-ADK (Agentic Development Kit) is a Go CLI tool that installs the Autopus harness into multiple coding CLI platforms. It uses an adapter-based architecture for platform abstraction and provides SPEC management, decision tracking, architecture analysis, and multi-model orchestration.

## Architecture Rules

- `internal/cli` depends on `pkg/*` only (never reverse)
- `pkg/*` packages do not depend on each other (except `pkg/template` as utility)
- `cmd/` contains only the entry point, no business logic
- Platform-specific code lives exclusively in `pkg/adapter/{platform}/`
